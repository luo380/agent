阶段 1：聊天主链路技术：FastAPI、SQLAlchemy、JWT、SSE、OpenAI 兼容接口
结果：能注册、登录、发消息、流式输出、保存消息
你学到：LLM 基础调用、后端 API、流式响应
求职关键词：LLM chat、streaming response、FastAPI

阶段 2：Agent 配置化技术：配置驱动设计
结果：一个 Agent 有 name / system_prompt / model / temperature / welcome_message
你学到：配置和运行时分离、系统提示词、多租户基础、会话和 Agent 关系
求职关键词：AI Agent、multi-tenant、prompt configuration

阶段 3：运行轨迹技术：Run / RunStep、状态机、日志设计
结果：知道一次消息执行了哪些步骤、哪一步失败
你学到：可观测性、审计日志 vs 业务日志、执行链路
求职关键词：observability、trace、run step

阶段 4：RAG 底层实现技术：文档解析、chunking、embedding、vector search、rerank
结果：上传文档后，回答能引用知识库内容
你学到：RAG 真正的核心链路
求职关键词：RAG、embedding、vector database、retrieval


阶段 5：LangChain 版 RAG技术：LangChain、Document Loader、Text Splitter、Embeddings、Retriever、Chain
结果：把你第 4 阶段手写的 RAG，再用 LangChain 重构一版
你学到：框架抽象和底层实现的映射关系
求职关键词：LangChain、RetrievalQA、Runnable、LCEL
这一步很重要，因为很多岗位 JD 会直接写 LangChain

阶段 6：工具调用技术：function calling、tool schema、HTTP 安全、超时重试
结果：模型能决定什么时候调用工具
你学到：Agent 工具链、ReAct 思路、安全边界
求职关键词：tool calling、function calling、agent tools

阶段 7：LangGraph / Agent Workflow技术：LangGraph、状态图、节点编排、条件分支
结果：把“聊天、RAG、工具调用、记忆”串成工作流 Agent
你学到：真正适合面试展示的 Agent orchestration
求职关键词：LangGraph、workflow orchestration、stateful agent

阶段 8：记忆与评测技术：memory、summary memory、evaluation、prompt/version 管理
结果：多轮对话更稳，能评估答案质量
你学到：短期记忆、长期记忆、上下文压缩、评测方法
求职关键词：memory、evaluation、prompt engineering
